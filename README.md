# @2e32/react-button

React button.

## How install

```bash
yarn add @2e32/react-button
```

## Usage

App.js

```javascript
// Once (css import)
import '@2e32/react-button/css';

const App = () => <Page />;

export default App;
```

Page.js

```javascript
import Button from '@2e32/react-button';

const Page = () => {
  return <Button>Click me!</Button>;
};

export default Page;
```
